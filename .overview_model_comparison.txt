┌─────────────────────────────────────────────────────────────────────────────┐
│ WISSENSCHAFTLICHE METHODIK                                                   │
└─────────────────────────────────────────────────────────────────────────────┘

   ✓ Reproduzierbarkeit
     - Fixed Random Seed (13)
     - Gleiche Evaluation-Parameter
     - Gleiche Test-Daten

   ✓ Statistische Robustheit
     - Paired t-tests (innerhalb Assets)
     - 95% Konfidenzintervalle
     - Autokorrelationskorrektur
     - Effektive Stichprobengröße

   ✓ Time-Series spezifisch
     - Diebold-Mariano Test
     - Rolling Window Validation
     - Kein Look-Ahead Bias


┌─────────────────────────────────────────────────────────────────────────────┐
│ METRIKEN                                                                     │
└─────────────────────────────────────────────────────────────────────────────┘

   1. TIME-SERIES METRIKEN (innerhalb Assets)
      • IC (Pearson): Lineare Korrelation von Returns
      • RankIC (Spearman): Rang-Korrelation (robust gegen Outlier)
      • MAE, RMSE: Forecast-Fehler auf Preisebene

   2. CROSS-SECTIONAL METRIKEN (über Assets)
      • Cross-Sectional RankIC: Ranking-Fähigkeit zwischen Assets
      • Wichtigste Metrik für Trading-Strategien!

   3. STATISTISCHE TESTS
      • Paired t-test: Vergleich über Assets
      • Diebold-Mariano: Forecast-Genauigkeit mit Autokorrelation
      • Konfidenzintervalle: Unsicherheit quantifizieren


┌─────────────────────────────────────────────────────────────────────────────┐
│ INTERPRETATION                                                               │
└─────────────────────────────────────────────────────────────────────────────┘

   ✅ Fine-Tuning ERFOLGREICH wenn:
      • p-Wert < 0.05 (statistisch signifikant)
      • Positive Differenz (Fine-Tuned > Zero-Shot)
      • 95% CI enthält keine 0

   ⚠️  Fine-Tuning NICHT erfolgreich wenn:
      • p-Wert ≥ 0.05 (nicht signifikant)
      • Differenz nahe 0
      • 95% CI überlappt stark
